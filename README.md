# hello Eventline

This is a *Hello-world* like project for
[Eventline](https://github.com/surebot/Eventline) project.

This project is a simplest REPL.

```bash
$ node .
# > help
# What? Use "hi", "hello" or "ping" messages.
# > hi
# Hi
# > ping
# pong
# >
```

## Features

This project provides the following features:

- respond with `hi`
- respond to `ping`
- show help otherwise
